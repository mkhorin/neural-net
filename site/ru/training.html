Обучение нейросети

Нейронные сети обучаются непосредственно в браузере.

Обучающий набор данных содержит несколько десятков тысяч образцов рукописных цифр, перемешанных случайным образом. Задаваемые на форме начальные параметры влияют на результат обучения.

  • Количество нейронов. Параметр отвечает за число нейронов в скрытом слое. Чем больше нейронов, тем дольше обучается сеть. Минимальное значение - 4, максимальное - 32.
  • Скорость обучения. Множитель изменений при схождении функции обучения к минимуму ошибок. Минимальное значение - 0.01, максимальное - 1.
  • Начальный порог ошибки. Если ошибка случайного начального состояния сети превышает заданное значение, то сеть будет инициализирована повторно. Минимальное значение - 0.01, максимальное - 1.
  • Конечный порог ошибки. Если разница между ошибками итераций обучения меньше заданного порога, то обучение прекращается. Минимальное значение - 0.0002, максимальное - 0.01.
  • Минимальный / максимальный начальный вес. Граничные значения при генерации начального состояния связей нейронной сети.
  • Сдвиг скрытого / выходного слоя. Смещение результата функции активации нейрона.

Обученную сеть можно сохранить либо локально, либо удаленно на сервер (при наличии прав). Локально хранимые сети доступны только в текущем браузере. Удаленные сети доступны всем пользователям.
